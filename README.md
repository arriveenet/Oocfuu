# 誕生日おめでとうございます！
本当は何かゲームを作りたかったのですが、時間と技術不足によりただ音楽が流れるしょうもないアプリになりました。  
もし、バグがあったり動かなかったりした場合は連絡してください。
# 使用
スペースキーを押すとHappy Birthday to youが流れる  
↓  
タイフーンパレードが流れる  
↓  
花火が3回上がる

# キーバインド
* aキー 左へ移動 
* dキー 右へ移動
* スペースキー　ジャンプ
* rキー　タイフーンパレードを最初から再生
* kキー　タイフーンパレードを一時停止
* oキー
